## Chat App

## Sobre

O ChatApp é uma plataforma de chat que utiliza o socket.io para a criação de salas e usuários.

## Backend

##### Instalando as dependências

```sh
yarn install
```

##### Rodando a aplicação local (dev)

```sh
yarn start
```

## Frontend

##### Instalando as dependências

```sh
yarn install
```

##### Rodando a aplicação local

```sh
yarn dev
```
